Privacy Policy

AEO Dispatch System

This Privacy Policy explains how the AEO Dispatch System collects, uses, stores, and protects user information.

Information We Collect

  • CNIC (for authentication)
  • User profile information
  • Google Account email (only after user authorization)
  • Generated reports and dispatch records
  • Google Drive file identifiers
  • Email delivery logs

Google Authorization

Users voluntarily authorize access to their Google account using Google OAuth 2.0. The authorization is used solely for:

  • Uploading generated reports to the user's own Google Drive.
  • Sending official dispatch emails from the user's own Gmail account.

The application never accesses data beyond the permissions granted by the user.

Data Storage

Application data is securely stored in Supabase. OAuth tokens are encrypted and protected using industry-standard security practices.

Use of Information

User information is used exclusively for operating the AEO Dispatch System, generating reports, storing files, and sending official communications.

Data Sharing

The application does not sell, rent, or share personal information with third parties except where required by law.

Security

Reasonable administrative and technical safeguards are implemented to protect user information against unauthorized access, alteration, or disclosure.

User Rights

Users may revoke Google access at any time through their Google Account security settings.

Contact

For questions regarding this Privacy Policy, contact the system administrator.
